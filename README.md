# Card Proxying Helper

A browser addon meant to ease some of the repititive aspects of collecting and managing proxy art images for Proxyshop.

## Features

Allows copying a card's details to clipboard from Scryfall's card page in a form that can be used as such or with minor adjustments as a filename for an image that is meant to be fed to Proxyshop. Also allows specifying custom text templates to modify the format of the copied text.

## Development environment

Install dependencies.
```sh
npm install
```

Run development build. This opens Chrome by default. See `package.json` for other options.
```sh
npm run dev
```

Create an installable zip.
```sh
npm run build
```
